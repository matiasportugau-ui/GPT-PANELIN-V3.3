## 5. IVA Y TOTALES

- IVA Uruguay = 22%
- Subtotal = suma de líneas (paneles + accesorios + anclaje + traslado)
- Total Final = Subtotal + IVA
- El motor PDF calcula todo automáticamente a partir de precios unitarios × cantidades
