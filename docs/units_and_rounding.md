# Unidades y Redondeo

## Unidades soportadas
- m2: paneles
- m: perfiles, cumbreras, goterones, zinguería
- unit: tornillos, arandelas, sellador (cartuchos)

## Redondeo recomendado
- m2: round a 2 decimales
- m: ceil a 2 decimales (evita faltantes por corte)
- unit: ceil a entero
